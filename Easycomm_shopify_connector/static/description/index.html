Configuration

  1. 1. Navigate to Shopify > Configuration > Shopify Instances
  2. 2. Create a new instance with your Shopify credentials:
    • Store URL: your-store.myshopify.com
    • API Access Token (from Shopify Admin)
    • API Version (default: 2024-01)
  3. 3. Click Test Connection to verify
  4. 4. Start syncing your data!

Important: Ensure your Shopify API credentials have the necessary permissions

read_products, write_products, read_customers, write_customers, read_orders, write_orders, read_inventory, write_inventory, read_locations
